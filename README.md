# Warehouse Inventory Management System

###Project Description
This is a simple Java-based Warehouse Inventory Management System that allows users to manage inventory using the Java Collections Framework (HashMap).

##  Features
- Add new items
- Remove items
- Update item quantity
- Search items by ID or Name
- Display all inventory items

##  How to Run
1. Compile all Java files:
   javac *.java

2. Run the program:
   java Warehouse
